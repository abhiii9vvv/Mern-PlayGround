# 🚀 MERN Playground

A comprehensive collection of **mini projects** to master **Frontend, Backend, and Full-Stack Development**.

## 🎯 What's Inside?

- **DOM & Browser APIs** - Event handling, scroll detection, online/offline status
- **UI Components** - Modals, toasts, carousels, accordions, dropdowns
- **JavaScript Fundamentals** - Arrays, strings, objects, algorithms, functions
- **React Magic** - Hooks, state management, context API, custom hooks
- **Express APIs** - REST endpoints, middleware, error handling
- **MongoDB & MERN** - Authentication, CRUD operations, JWT, database design

## 💡 How to Use?

Each folder contains **individual tasks** with TODO comments. Pick any task and implement it step by step. Perfect for **learning, interviews, and hands-on practice**.

## 🔥 Features

✅ 55+ beginner to intermediate challenges  
✅ Real-world scenarios  
✅ No frameworks bloat - vanilla JS & React
